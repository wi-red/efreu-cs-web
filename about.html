About this project

The project

A web project for course TI402

This website was built as part of the TI402 Web Development course of the second-year Bachelor in Computer Science at EFREI Paris. The goal of the project is to apply the concepts of HTML5, CSS and JavaScript covered in the lab sessions to a complete, multi-page website.

The chosen subject is a showcase site for the Computer Science Department itself. It presents the programs, the faculty and the research labs in a single place, in a layout meant to look like a small institutional website.

What we did

  • Designed a mockup of the website.
  • Wrote, W3C-valid HTML5 for four main pages.
  • Styled the website with a single, organised CSS file using the EFREI brand colours.
  • Implemented a responsive layout with media queries for tablet and mobile.
  • Added a contact form and several data tables to demonstrate forms and tables manipulation.

The constraint of building everything without any framework or library was a useful discipline. It forced us to write CSS from scratch, to think about reusable classes and to keep the code easy to read. We chose not to use any javascript because of time issues.

Tech and tools used

  • HTML5 with semantic elements (header, nav, main, section, article, footer).
  • CSS with Flexbox, CSS Grid and media queries.
  • Git for version control and collaboration.
  • Canva for the wireframe.
